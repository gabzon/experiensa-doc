4. Custom fields

Custom fields are added in the piklist folder
  • 4.1 General
  • 4.2 Flight information
  • 4.3 Accomodation information
  • 4.4 Itinerary information

4.1 General


 
Label Field type Field name Comments
Price text product_price
Currency text product_currency To use if it is different from the main currency
Expiry date DatePicker product_expiry_date If today is > than expiry date then change product status to inactive
Sell to other agencies checkbox product_is_tour_operator When check display percentage field
Percentage(%) number product_tour_operating_percentage This field is invisible, it will be display with the condition that "sell to other agencies" is checked
Slogan text product_slogan
Additional information & Conditions textarea product_information_conditions
Photo Gallery File Upload product_photos
Flight checkbox product_has_flight When check display flight group
Accommodation checkbox product_has_accommodation When check display accommodation group
Itinerary checkbox product_has_itinerary When check display itinerary group

4.2 Flights information


 
Label Field type Field name Comments
Airline text product_airline
Flight number text product_flight_number
Class Select box product_flight_class Options:(Economy, First, Business). Default:Economy
Departure city text product_departure_city
Departure date DatePicker product_departure_date
Departure time text product_departure_time
Arrival city text product_arrival_city
Arrival date DatePicker product_arrival_date format dd/mm/yyyy
Arrival time text product_arrival_time validate format hh:mm, 24h system
Hand bag quantity Select box product_handbag_qty (values : [0,1, 2], 1 = default)
hand bag weight number product_handbag_weight validate is number
Luggage quantity Select box product_luggage_qty (0,1, 2)(1 = default)
Luggage weight number product_luggage_weight validate: is number
Refundable Checkbox product_refund
Flexible Checkbox product_flexible
Coments Text product_comments

4.3 Accomodation information


 
Label Field type Field name Comments
Etablishment name text product_etablishment_name
Etablishment type Select box product_etablishment_type values: hotel, b&b, residence, private apartment, airbnb
Rating(stars) Select box product_etablishment_rating values: 1-5
Check-in Date DatePicker product_etablishment_checkin_date format dd/mm/yyyy
Check-in Time text product_etablishment_checkin_time
Check-out Date DatePicker product_etablishment_checkout_date (format dd/mm/yyyy)
Check-out Time text product_etablishment_checkout_time validate time
Photo Gallery File Upload product_etablishment_photos
Comments textarea product_etablishment_comments
